中文phpBB-WAP
=========

phpBB-WAP 中文版取自于 phpbb-wap.ru 中的 phpBB-WAP v8 版本，而 phpBB-WAP 是世界上知
名的 phpBB 论坛开源软件的移动终端版本。phpBB-WAP 这个名字，
是 PHP Bulletin Board Wireless Application Protocol 的缩写。
