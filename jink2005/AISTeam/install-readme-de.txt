### Voraussetzungen ###
Server:
- PHP 5.1 oder neuer
- MySQL 4 oder neuer

Client:
- Firefox 3, Internet Explorer 7/8, Opera 9, Safari
- Cookies aktiviert
hnjhnhnjhhh

### Installationsanweisungen ###
1. Entpacke das 2-plan Archiv.
2. Lade alle Dateien und Ordner auf deinen Server hoch.
3. Mache die folgenden Dateien und Ordner beschreibbar (chmod 777):
- /templates_c
- /files
- /config/standard/config.php
4. Erzeuge eine neue MySQL-Datenbank.
5. Öffne install.php im Browser.
6. Wenn die Installation erfolgreich war, lösche install.php und update.php.


### Update ###
   1. Entpacke das 2-plan Archiv.
   2. Lade die bestehende config.php vom Server runter.
   3. Kopiere die heruntergeladene config.php in den Ordner /config/standard/
   4. Lade alle Dateien und Ordner hoch.
   5. Öffne update.php im Browser.
   6. Wenn das Update erfolgreich war, lösche install.php und update.php.
