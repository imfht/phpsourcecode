Piano-MIDI
===

接入电钢琴等设备MIDI接口，通过钢琴的弹奏实时显示瀑布流线条。[在线演示DEMO](https://piano.openadc.com/index.php/index/index/showView)

功能特性
=======

- 实时接收MIDI数据并显示
- 可调控的速度和显示的颜色

环境依赖
=======
- [ThinkPHP](https://github.com/top-think/think)
- [JQuery](https://github.com/jquery/jquery)
