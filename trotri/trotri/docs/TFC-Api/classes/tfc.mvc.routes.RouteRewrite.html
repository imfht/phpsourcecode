RouteRewrite

Extends \tfc\mvc\routes\Route

RouteRewrite class file 标准路由

一.标准路由例子： URL：http://domain.com/author/iphper

$route = new RouteRewrite(
    'author/:username',
    array(
        'controller' => 'profile',
        'action'     => 'userinfo'
    )
);
$value = array(
    'controller' => 'profile',
    'action'     => 'userinfo',
    'username'   => 'iphper'
);

二.规则匹配： URL：http://domain.com/archive/2012

$route = new RouteRewrite(
    'archive/:year',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        'year'    => '\d+'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    'year'       => '2012'
);
如果URL：http://domain.com/archive/test，将不匹配，并且尝试匹配下一个路由

三.有缺省值时，规则匹配： URL：http://domain.com/archive

$route = new RouteRewrite(
    'archive/:year',
    array(
        'year'       => '2012',    // 缺省值
        'controller' => 'archive',
        'action'     => 'show'
    )
);
$value = array(
    'year'       => '2012',
    'controller' => 'archive',
    'action'     => 'show'
);

四.模仿模块路由： URL：http://domain.com/passport/profile/userinfo/username/iphper

$route = new RouteRewrite(
    ':module/:controller/:action/*',
    array(
       'module' => 'default'    // 缺省值
    )
);
$value = array(
    'module'     => 'passport',
    'controller' => 'profile',
    'action'     => 'userinfo',
    'username'   => 'iphper'
);
如果URL：http://domain.com/archive/test，将不匹配，并且尝试匹配下一个路由
